# 🌐 Projeto HTML e CSS - Curso Alura

Este é um projeto simples desenvolvido como parte do curso de **HTML e CSS** da [Alura](https://www.alura.com.br/). O objetivo foi praticar a estruturação de páginas web utilizando HTML e estilização com CSS.

## 📝 Sobre o projeto

O projeto inclui os seguintes elementos fundamentais de uma página web:

- Header: Cabeçalho da página.
- Nav: Barra de navegação.
- Main: Conteúdo principal.
- Footer: Rodapé da página.
- Imagens: Utilização de imagens para compor o design da página.
- Textos: Inclusão de conteúdos textuais.

## 📷 Fontes das imagens e textos

- Imagens: Foram retiradas da internet.
- Textos: O conteúdo textual referente à parte de HTML também foi obtido de fontes online.

## 🛠 Tecnologias utilizadas

- HTML5
- CSS3
- Visual Studio Code

   git clone https://github.com/seu-usuario/nome-do-repositorio.git

2. Abra o arquivo `index.html` em um navegador.
